Causality Tests¶

The gtime.causality module deals with the causality tests for time series data.

causality.ShiftedLinearCoefficient([…])

Test the shifted linear fit coefficients between two or more time series.

causality.ShiftedPearsonCorrelation([…])

Class responsible for assessing the shifted Pearson correlations (PPMCC) between two or more series.

causality.GrangerCausality(target_col, x_col)

Class to check for Granger causality between two time series, i.e.

Compose¶

The gtime.compose module contains meta-estimators for building composite models with transformers.

compose.FeatureCreation(transformers[, …])

Applies transformers to columns of a pandas DataFrame.

Feature Extraction¶

The gtime.feature_extraction module deals with the creation of features starting from a time series.

feature_extraction.Shift([shift])

Perform a shift of a DataFrame of size equal to shift.

feature_extraction.MovingAverage([window_size])

For each row in time_series, compute the moving average of the previous window_size rows.

feature_extraction.MovingCustomFunction(…)

For each row in time_series, compute the moving custom function of the previous window_size rows.

feature_extraction.Polynomial([degree])

Compute the polynomial feature_extraction, of a degree equal to the input degree.

feature_extraction.Exogenous

Reindex exogenous_time_series with the index of time_series.

feature_extraction.CustomFeature(func, **kwargs)

Constructs a transformer from an arbitrary callable.

feature_extraction.Detrender(trend, trend_x0)

Apply a de-trend transformation to a time series.

feature_extraction.SortedDensity([…])

For each row in time_series, compute the sorted density function of the previous window_size rows.

feature_extraction.CrestFactorDetrending([…])

Crest factor detrending model.

Feature Generation¶

The gtime.feature_generation module deals with the creation of features that do not depend on the input data, but just on its index.

feature_generation.PeriodicSeasonal([…])

Create a sinusoid from a given date and with a given period and amplitude.

feature_generation.Constant([constant, length])

Generate a pd.DataFrame with one column, of the same length as the input X and containing the value constant across the whole column.

feature_generation.Calendar([region, …])

Create a feature based on the national holidays of a specific country.

Forecasting¶

The gtime.forecasting module contains a collection of machine learning models, for dealing with time series data.

forecasting.GAR(estimator[, explainer_type, …])

Generalized Auto Regression model.

forecasting.GARFF(estimator[, explainer_type])

Generalized Auto Regression model with feedforward training.

forecasting.MultiFeatureGAR(estimator[, …])

Generalized Auto Regression model.

forecasting.TrendForecaster(trend, trend_x0)

Trend forecasting model.

forecasting.HedgeForecaster([learning_rate, …])

Regressor model using Hedge algorithm.

forecasting.NaiveForecaster

Naïve model, all predicted values are equal to the most recent available observation.

forecasting.SeasonalNaiveForecaster(…)

Seasonal naïve model.

forecasting.DriftForecaster

Simple drift model, calculates drift as the difference between the first and the last elements of the train series, divided by the number of periods.

forecasting.AverageForecaster

Predicts all future data points as an average of all train items and all test items prior to is

forecasting.MultiFeatureMultiOutputRegressor(…)

Multi target regression with option to choose the features for each target.

Hierarchical¶

The gtime.hierarchical module contains hierarchical time series models.

hierarchical.HierarchicalBase(model[, …])

Base class for hierarchical models.

hierarchical.HierarchicalNaive(model)

Simplest hierarchical model possible.

hierarchical.HierarchicalBottomUp(model[, …])

Hierarchical model with prediction following the Bottom-Up procedure.

hierarchical.HierarchicalTopDown(model[, …])

Hierarchical model with prediction following the Top-Down procedure.

hierarchical.HierarchicalMiddleOut(model[, …])

Hierarchical model with prediction following the Top-Down procedure.

Metrics¶

The gtime.metrics module contains a collection of different metrics.

metrics.non_zero_smape(y_true, List, …)

Compute the ‘Symmetric Mean Absolute Percentage Error’ (SMAPE) between two vectors without considering the 0 in the true values.

metrics.smape(y_true, List, numpy.ndarray], …)

Compute the ‘Symmetric Mean Absolute Percentage Error’ (SMAPE) between two vectors.

metrics.max_error(y_true, List, …)

Compute the maximum error between two vectors.

metrics.mse(y_true, List, numpy.ndarray], …)

Compute the Mean Squared Error(MSE) between two vectors.

metrics.log_mse(y_true, List, …)

Compute the Mean Squared Log Error(MSLE) between two vectors.

metrics.r_square(y_true, List, …)

Compute the R squared (Coefficient of Determination) between two vectors.

metrics.mae(y_true, List, numpy.ndarray], …)

Compute the Mean Absolute Error(also called, Mean Absolute Deviation(MAD) or Mean Ratio) between two vectors.

metrics.mape(y_true, List, numpy.ndarray], …)

Compute the Mean Absolute Percentage Error(MAPE) between two vectors.

metrics.rmse(y_true, List, numpy.ndarray], …)

Compute the Root Mean Squared Error(RMSE) between two vectors.

metrics.rmsle(y_true, List, numpy.ndarray], …)

Compute the Root Mean Squared Log Error(RMSLE) between two vectors.

metrics.gmae(y_true, List, numpy.ndarray], …)

Compute the Geometric Mean Absolute Error between two vectors.

Model Selection¶

The gtime.model_selection module deals with model selection.

model_selection.FeatureSplitter([drop_na_mode])

Splits the feature matrices X and y in X_train, y_train, X_test, y_test.

model_selection.horizon_shift(time_series, …)

Perform a shift of the original time_series for each time step between 1 and horizon.

model_selection.time_series_split(time_series)

Time Series cross-validator

model_selection.blocking_time_series_split(…)

Time Series cross-validator

Plotting¶

plotting.seasonal_plot(df, cycle, freq, …)

Seasonal plot function

plotting.seasonal_subplots(df, cycle, freq, …)

Seasonal subplots, a series of subplots representing average values and cycle-over-cycle dynamics or box plots for each season.

plotting.acf_plot(df, max_lags, ci, partial)

ACF plot function, showing autocorrelation or partial autucorrelation for lags up to max_lags.

plotting.lag_plot(df, lags, List[int]], …)

Lag plots, scatter plot of x_i against x_i-k for every k in lags.

Preprocessing¶

The gtime.preprocessing module deals with the preprocessing of time series data.

preprocessing.TimeSeriesPreparation([start, …])

Transforms an array-like sequence in a period-index DataFrame with a single column.

Regressors¶

The gtime.regressors module contains regression models.

regressors.LinearRegressor([loss])

Implementation of a LinearRegressor that takes a custom loss function.

regressors.MultiFeatureMultiOutputRegressor(…)

Multi target regression with option to choose the features for each target.

regressors.ExplainableRegressor(estimator, …)

Wraps the most commons scikit-learn regressor to offer a nice to use interface to fit/predict models and at the same time to explain the predictions.

Time series models¶

The gtime.time_series_models module contains time series models.

time_series_models.TimeSeriesForecastingModel(…)

Base class for a generic time series forecasting model.

time_series_models.AR(p, horizon[, …])

Standard AR model for time series

time_series_models.Naive(horizon)

Naive model pipeline, no feature creation and NaiveModel() as a model

time_series_models.SeasonalNaive(horizon, …)

Seasonal naive model pipeline, no feature creation and SeasonalNaiveModel() as a model

time_series_models.Average(horizon)

Average model pipeline, no feature creation and AverageModel() as a model

time_series_models.Drift(horizon)

Simple drift model pipeline, no feature creation and DriftModel() as a model

time_series_models.CVPipeline(models_sets[, …])

Cross-validation for models of time_series_models classes
